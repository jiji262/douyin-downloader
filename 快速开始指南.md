# 抖音下载器 (Douyin-Downloader) 常用运行命令指南

本文档记录了运行本项目以及使用本地 Whisper 进行语音转文字的常用命令。

## 1. 运行抖音下载器项目

在配置好环境依赖及 `config.yml` 之后，在项目根目录运行以下命令以启动下载任务：

### 基础运行 (默认使用 config.yml)
```bash
python run.py
```

### 指定配置文件运行
如果你有多个配置文件（例如针对不同博主），可以使用 `-c` 参数：
```bash
python run.py -c config.yml
```
> **注意**：`python run.py` 在不加参数时，默认也会寻找当前目录下的 `config.yml`。使用 `-c` 主要是为了在有多个配置（如 `config_personal.yml`, `config_work.yml`）时进行切换。

## 2. 启动 Web UI 管理界面

Web UI 提供可视化的配置编辑、任务控制和本地资产管理。需要同时启动**后端 API** 和**前端开发服务器**。

```bash
.venv\Scripts\python.exe -m uvicorn api.server:app --reload --host 127.0.0.1 --port 8000
```

## 3. 使用本地 Whisper 进行语音转文字

本项目已支持从 `config.yml` 读取默认配置。推荐在 `config.yml` 末尾添加 `local_whisper` 配置块。建议将模型设置为 `turbo`（对应本地已有的 `large-v3-turbo` 模型），这样运行速度最快且无需下载。

### 配置文件设置参数说明
在 `config.yml` 末尾添加或修改：
```yaml
local_whisper:
  model: turbo            # 模型名称。可选: tiny, base, small, medium, large, turbo
  sc: true                # 是否繁体转简体。需先执行 pip install opencc-python-reimplemented
  srt: true               # 是否生成 SRT 字幕文件
  skip_existing: true     # 是否跳过已处理的文件
```

### 极简运行 (推荐)
设置好 `config.yml` 后，直接运行脚本，它会自动读取上述配置：
```bash
python cli/whisper_transcribe.py
```

### 手动覆盖配置运行
如果你想临时改变模型或关闭某项功能，可以使用命令行参数（这会覆盖配置文件里的设置）：

```bash
# 强制使用 turbo 模型 + 繁简转化 + 生成 srt 字幕
python cli/whisper_transcribe.py -m turbo --srt --sc

# 只识别单个特定文件
python cli/whisper_transcribe.py -f "视频路径.mp4"
```

### 参数详细说明
*   `-m [模型]`: 指定模型。本电脑已缓存 `tiny`, `base`, `large-v3-turbo` (参数请填写 `turbo`)。
*   `--srt`: 生成带时间轴的字幕文件（.srt）。
*   `--sc`: 繁体转简体。由于 Whisper 经常随机输出繁体，开启此项可统一为简体中文。
*   `--skip-existing`: 开启后会检查是否已有转录文件，跳过已处理的视频，极大提升批量处理速度。
